Installation Guide
  • Download the zip file.
  • Extract the file using it's password.
  • Disable internet connection.
  • Run ' install.bat ' to start the installation.
  • After the installation, use any of the below Methods to Activate MS Office :

    • Attention:

    - After running the ' install.bat ' you will not be able to stop the installation.

    - You can install "Visio" & "Project" by clicking on ' install-visio.bat ' and ' install-project.bat' .


    Method 1

    Use KMS Server:

  • Open CMD as an administrator then enter the following commands:

    • $ cd "C:\Program Files\Microsoft Office\Office16"
    $ cscript ospp.vbs /sethst:kms.digiboy.ir
    $ cscript ospp.vbs /act


    Attention:

    - Make sure that you are connected to the internet.

    Method 2

  • Use Microsoft Toolkit (Portable software) to activate Office.


    • System Requirements:

    - Windows 10, Windows Server 2019
    - Some features may require .NET 3.5 or 4.6 and higher to also be installed
